# PassForge — Password Generator

A clean, modern password generator built with Next.js and Tailwind CSS. Generate strong, customizable passwords instantly with real-time strength analysis and localStorage-backed history.

## Features

- **Generator** — configure length (4–64), character sets (upper, lower, numbers, symbols), and quick presets
- **Strength Meter** — 8-segment visual strength bar with entropy-based scoring
- **History** — saves up to 50 passwords in localStorage, filterable by strength level
- **Security Tips** — educational guide covering entropy, dos/don'ts, and best practices
- Copy to clipboard, one-click save, and instant regeneration

## Tech Stack

- [Next.js 15](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
